КОНТАКТНАЯ ИНФОРМАЦИЯ

Вариант №3

  • Добавить на страницу форму, оформленние полей которой зависит от корректности заполнения (valid/invalid) и обязательности.

Порядок выполнения работы

  1. Создать простой HTML-документ, содержащий блочные элементы (они понадобятся для демонстрации использования некоторых стилей).
  2. Показать в созданном документе три способа использования каскадных таблиц стилей:
    • inline-описание (используется атрибут style);
    • с помощью контейнера ‹style› в секции заголовка;
    • подключить внешним файлом;
  3. Используя селекторы CLASS и ID показать использование стилей для форматирование шрифта, текста, изменения фона документа.
  4. Использовать абсолютное и относительное позиционирование.
  5. Показать использование плавающей модели.
  6. Проверить отображение созданного документа в разных браузерах.
